call, apply, bind

Это три метода, которыми мы сами назначаем this — не полагаясь на объект перед точкой. Их называют «явной привязкой», и в приоритете правил они сильнее обычного вызова через точку.

call и apply — вызвать сразу с нужным this

Оба вызывают функцию немедленно и первым аргументом принимают объект для this. Разница только в том, как передавать остальные аргументы:

  • call — аргументы списком через запятую;
  • apply — аргументы одним массивом (удобная мнемоника: apply → array).

bind — создать функцию с привязанным this

bind ничего не вызывает. Он возвращает новую функцию, у которой this навсегда привязан к указанному объекту. Её можно сохранить и вызвать когда угодно — this не потеряется:
