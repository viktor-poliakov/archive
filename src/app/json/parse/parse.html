JSON.parse — разбор строки

JSON.parse делает обратное stringify: берёт строку JSON и превращает её в настоящее значение JavaScript, с которым уже можно работать. Это «распаковщик» посылки — из текста снова получаем объект.

Базовое использование

На входе — строка, на выходе — обычный объект (или массив, число, строка…). К результату сразу можно обращаться по ключам и индексам:
