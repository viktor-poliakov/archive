Объекты: основы

Объект в JavaScript — это набор именованных значений, сложенных вместе. Если переменная хранит один кусочек данных (число, строку), то объект хранит сразу много связанных кусочков под понятными именами.

Аналогия: объект — это анкета или карточка товара. У карточки есть поля: «название», «цена», «в наличии». Каждое поле (его называют ключ или свойство) хранит своё значение.

Литерал объекта

Чаще всего объект создают литералом — фигурными скобками {}, внутри которых перечислены пары ключ: значение через запятую:

Значением свойства может быть что угодно: число, строка, булево, массив, другой объект и даже функция (тогда свойство называют методом).

Доступ к свойствам: точка и скобки

Есть два способа прочитать или записать свойство:

  • через точку — user.name — короткий и привычный способ, когда имя свойства известно заранее;
  • через квадратные скобки — user['name'] — нужен, когда имя свойства лежит в переменной или содержит символы, которые нельзя написать после точки (пробел, дефис, начинается с цифры).
