DOM — основы

DOM (Document Object Model) — это представление страницы в виде дерева объектов. Браузер, прочитав HTML, строит из него дерево, а JavaScript работает уже с этим деревом, а не с текстом HTML. Меняешь дерево — меняется то, что видит пользователь.

Аналогия — генеалогическое дерево. У страницы есть общий предок (<html>), у него дети (<head> и <body>), у них — свои дети, и так вглубь. Каждый «человек» в этом дереве — узел, к которому можно обратиться из кода: найти, прочитать, изменить, удалить. DOM — это и есть такое живое дерево страницы.

Узлы и элементы — не одно и то же
