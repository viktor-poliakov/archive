IndexedDB

IndexedDB — это полноценная база данных прямо в браузере. Её берут, когда localStorage уже мал: нужно хранить много данных, целые объекты (а не строки) и работать с ними асинхронно, не подвешивая страницу.

Аналогия — архив с картотекой. Если localStorage — ящик стола, то IndexedDB — целый архив: внутри базы есть «шкафы» (хранилища объектов), в них лежат карточки-записи, а работа идёт строго по регламенту — через транзакции.

Когда нужна

  • данных много — десятки мегабайт и больше;
  • хранить нужно объекты и файлы, искать по ним;
  • приложение должно работать офлайн (кэш, черновики);
  • для простой пары «ключ-значение» она избыточна — берите localStorage.
