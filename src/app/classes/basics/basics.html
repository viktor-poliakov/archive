Классы: основы

Часто нужно много однотипных объектов: пользователи, товары, кнопки. У всех одинаковый набор данных и поведение, отличаются только значения. Описывать каждый вручную — утомительно и легко ошибиться. Класс решает это: он задаёт «форму» объекта один раз, а дальше по ней создаются экземпляры.

Аналогия — чертёж. Класс — это чертёж, который вы сами нарисовали: на нём отмечено, какие у объекта данные (поля) и что он умеет (методы). По одному чертежу можно построить сколько угодно объектов; у каждого свои значения, но устройство общее. Никакой магии — вы полностью описываете чертёж сами.

Объявление класса

Класс объявляют словом class. Внутри — специальный метод constructor (готовит новый объект), обычные методы и поля:

  • constructor вызывается автоматически при new User(...) и заполняет объект начальными данными.
  • Поля (role = 'guest') — данные, свои у каждого экземпляра.
  • Методы (greet) — поведение, общее для всех экземпляров.
  • this внутри — это создаваемый объект.
