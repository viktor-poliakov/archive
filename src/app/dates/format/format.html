Форматирование и парсинг

Дату нужно то показать человеку («26 января 2024»), то сохранить для машины («2024-01-26»), то наоборот — прочитать из строки. Разберём оба направления: как выводить дату и как разбирать её из текста.

Для человека: toLocale…

Методы toLocale… выводят дату по правилам локали (языка и региона), а объект опций настраивает, что именно показать:

Intl.DateTimeFormat

Разбор строк

Обратная задача — получить Date из строки. Здесь важное правило: надёжен только ISO-формат. Остальные форматы разные браузеры разбирают по-своему, поэтому на них полагаться нельзя:

Invalid Date: если строку не удалось разобрать, получится не ошибка, а «невалидная дата». Проверяйте через Number.isNaN(date.getTime()).

Что когда использовать

Способ Для чего Пример результата
toLocaleString(...) показать человеку 26 января 2024 г.
Intl.DateTimeFormat показывать часто (быстрее) 26 января 2024 г. в 14:30
toISOString() хранить / API (UTC) 2024-01-26T14:30:00.000Z
new Date('...ISO') разобрать из строки Date из '2024-01-26'
