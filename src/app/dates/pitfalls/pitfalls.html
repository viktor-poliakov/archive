Кратко

  • Внутри — timestamp (мс от 1970); разница дат = вычитание.
  • Месяцы 0–11; getDate() (число) ≠ getDay() (день недели, 0 = вс).
  • Date изменяем — копируйте через new Date(d).
  • Парсите только ISO; помните про UTC-сдвиг у 'YYYY-MM-DD'.
  • Локализованный вывод — через Intl.DateTimeFormat.

Многое из этого — «врождённые» проблемы Date. Их решает новый стандарт: неизменяемые объекты, месяцы с 1, нормальные таймзоны — см. Temporal API.
